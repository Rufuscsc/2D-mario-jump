  ## Key Features
  
  * Walking 
    - press the arrow keys on your keyboard
  * Jumping
    - press spacebar
  * Enemies
    - to escape enemies you will have to jump over them  
  * Coins
    - Collect coins by running into it 
  * Levels
    - Maximum level is level 7
  
  ## How To Use
